# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

The `[Unreleased]` body is **generated** from `.changesets/*.md` by `scripts/regen-generated.sh`. Do not edit it by hand — add a per-PR changeset instead (see `.changesets/README.md`). The `block-generated-edits` CI job will fail any PR that touches this file directly. Released sections (`## [X.Y.Z] — date`) are written once by `scripts/release.sh` and never regenerated thereafter.
